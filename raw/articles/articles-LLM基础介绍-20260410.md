# LLM基础介绍
LLM（Large Language Model，大语言模型）是参数量达到数十亿甚至数百亿以上的人工智能模型，基于Transformer架构训练而成，核心能力包括自然语言理解、逻辑推理、内容生成和上下文记忆。

LLM的核心特征：
1.  参数量巨大：通常在数十亿以上，qwen3.5:27b-q4_K_M的参数量为270亿，属于中大型LLM；
2.  上下文理解能力：能理解长文本的语义和逻辑关系，适配知识管理场景；
3.  生成能力：能生成符合人类语言习惯的文本，可用于Wiki页面生成；
4.  泛化能力：能适配多种场景，包括资料解析、查询响应、内容总结。

常见应用场景：个人知识管理、学术论文解析、内容创作、智能查询等，本文主要用于LLM Wiki的资料摄入测试。
